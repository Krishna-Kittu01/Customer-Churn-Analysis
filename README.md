# Customer Churn Analysis

## Project Overview
This project analyzes customer subscription data to identify the key factors influencing customer churn. The analysis calculates important business metrics such as Customer Churn Rate, Customer Lifetime Value (CLV), and Monthly Recurring Revenue (MRR) Loss.

## Objectives
- Calculate Customer Churn Rate
- Calculate Customer Lifetime Value (CLV)
- Calculate Monthly Recurring Revenue (MRR) Loss
- Analyze churn by Contract Type
- Analyze churn by Payment Method
- Analyze churn by Customer Tenure
- Generate visualizations and business insights

## Tools & Technologies

## Key Findings
- Overall Churn Rate: **46.67%**
- Month-to-Month contracts show the highest churn.
- Long-term contracts have better customer retention.
- Higher monthly charges are associated with higher churn.
- Businesses can reduce churn through loyalty programs and annual subscription plans.

## Conclusion
This project demonstrates how customer churn analysis can help businesses improve customer retention using data-driven insights.
